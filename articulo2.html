HTML Y CSS

HTML y CSS son los pilares fundamentales para crear páginas web. Aunque son diferentes, trabajan juntos para hacer que una página web sea atractiva y funcional.

HTML (Lenguaje de Marcado de Hipertexto): Es el "esqueleto" de una página web. Piensa en él como el conjunto de instrucciones que le dice al navegador qué mostrar. Con HTML, se agregan cosas como títulos, párrafos, imágenes, enlaces y más. Es como organizar el contenido de una página.

CSS (Hojas de Estilo en Cascada): Es el "estilo" o "decoración" de una página web. Con CSS, puedes cambiar colores, tamaños de texto, márgenes y todo lo que tiene que ver con cómo se ve el contenido. Es como elegir la ropa de una persona y hacer que se vea bonita.

¿Cómo funcionan juntos? Primero, HTML crea la estructura de la página, como los bloques de texto y las imágenes.
Luego, CSS se usa para hacer que esos bloques se vean bonitos: cambiar colores, alineación, tamaños y muchas otras cosas.
